# Citation du Jour - TREVELIO

Bannière animée affichant une citation inspirante quotidienne pour les dirigeants de TPE.

## Utilisation

Ouvrir directement `citation_du_jour.html` dans un navigateur ou l'intégrer via iframe :

```html
<iframe src="https://votre-username.github.io/citation-du-jour/citation_du_jour.html" 
        width="100%" 
        height="44" 
        frameborder="0" 
        scrolling="no">
</iframe>
```

## Caractéristiques

- 30 citations inspirantes qui alternent chaque jour
- Animation de défilement fluide
- Design épuré aux couleurs de TREVELIO (#253985)
- Accessible et responsive
- Mode réduit pour les personnes sensibles au mouvement
